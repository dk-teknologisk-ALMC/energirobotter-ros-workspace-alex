# Face Detection

Face detection with YOLOv8 on the `/camera` topic.

Download the face-detection model
[yolov8n-face.pt](https://github.com/akanametov/yolov8-face/releases/download/v0.0.0/yolov8n-face.pt)
from the
[yolo-face repository](https://github.com/akanametov/yolo-face/tree/v0.0.0)
and place it in the `models/` directory.
